# 常用的 Unicode 字符

### 常用 ISO-8859-1/ISO-Latin1 字符
HTML 4.01 支持 ISO 8859-1 (Latin-1) 字符集。（实体名称对大小写敏感。）

| 结果 | 实体名称 | 实体编号 | 说明/备注 |
|:---:|:--------|:-------|:---------|
| -   | -       | \&#10; | 换行 |
| -   | -       | \&#13; | 回车 |
| "   | \&quot; | \&#34; | - |
| &   | \&amp;  | \&#38; | - |
| '   | \&apos; | \&#39; | - |
| <   | \&lt;   | \&#60; | - |
| >   | \&gt;   | \&#62; | - |
| ¥   | \&yen;  | \&#165; | yen，人民币符号 |
| ©   | \&copy; | \&#169; | copyright |
