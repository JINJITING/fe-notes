    # ECMAScript

    由 TC39 委员会制定标准。JavaScript 是 ECMAScript 的一种实现。

    • ES262 (opens new window)：JS 语言规范本身
    • ES402 (opens new window)：制定 ES5 和后面版本的国际化 API 标准
    • ES404：JSON 规范
    • ES414：规定了和 ECMAScript 相关的标准有哪些（即上述 3 条）
